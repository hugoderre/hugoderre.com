Voici un script Python qui utilise l'API Twitter pour tweeter "feur" en réponse à tous les tweets finissant par "quoi" :

import tweepy

# Remplacez les valeurs suivantes par vos propres clés d'API et tokens d'accès
consumer_key = "your_consumer_key"
consumer_secret = "your_consumer_secret"
access_token = "your_access_token"
access_token_secret = "your_access_token_secret"

# Créez une connexion à l'API Twitter
auth = tweepy.OAuth1UserHandler(consumer_key, consumer_secret, access_token, access_token_secret)
api = tweepy.API(auth)

# Créez une fonction qui sera appelée chaque fois qu'un nouveau tweet est publié
def tweet_feur(tweet):
  # Vérifiez si le tweet se termine par "quoi"
  if tweet.text.endswith("quoi"):
    # Répondez au tweet avec "feur"
    api.update_status("@" + tweet.user.screen_name + " feur", in_reply_to_status_id=tweet.id)

# Créez un "stream" qui écoute les tweets en temps réel
stream = tweepy.Stream(auth, tweet_feur)
stream.filter(track=["quoi"])

Il vous suffit de remplacer "your_consumer_key", "your_consumer_secret", "your_access_token" et "your_access_token_secret" par vos propres clés d'API et tokens d'accès obtenus sur le site de développeur de Twitter. Vous pouvez également personnaliser le filtre utilisé pour sélectionner les tweets à surveiller en modifiant la liste de mots-clés dans la ligne stream.filter(track=["quoi"]).

Notez que ce script enverra un tweet "feur" en réponse à chaque tweet finissant par "quoi" qui sera publié pendant que le script est en cours d'exécution. Si vous voulez tweeter "feur" uniquement en réponse aux tweets publiés par d'autres utilisateurs (et non à vos propres tweets), vous pouvez utiliser la méthode stream.filter(follow=["user_id"]) au lieu de stream.filter(track=["quoi"]), où "user_id" est l'identifiant de l'utilisateur à suivre.

Pour en savoir plus sur l'utilisation de l'API Twitter et de la bibliothèque tweepy en Python, vous pouvez consulter la documentation officielle de tweepy : https://tweepy.readthedocs.io/.
